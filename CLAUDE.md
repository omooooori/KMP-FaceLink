## Task Management

- **セッション開始時に [TASKS.md](./TASKS.md) を必ず読むこと**
- 次にやるべきタスク（status = TODO のうち上位のもの）を確認してから作業を開始する
- タスク完了時は以下を両方行う:
  1. `TASKS.md` の status を `DONE` に更新
  2. 対応する GitHub Issue を close (`gh issue close <number>`)
